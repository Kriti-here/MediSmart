# 💊 MediSmart — Drug Cost Comparison Tool

> A web-based tool that helps users compare the costs of medications across different sources or pharmacies, designed to support informed, budget-friendly healthcare decisions.

🌐 **Live Demo:** [drug-cost-comparison-tool-teal-ocean.reflex.run](https://drug-cost-comparison-tool-teal-ocean.reflex.run/)

---

## 📌 About

MediSmart empowers patients and caregivers to make smarter healthcare choices by making drug pricing transparent and easy to evaluate at a glance. Whether you're managing a chronic condition or looking for a one-time prescription, MediSmart helps you find the most affordable option without the hassle.

---

## ✨ Features

- 🔍 Search and compare drug prices across multiple sources/pharmacies
- 💰 Transparent cost breakdown for better financial decisions
- 📊 Clean, intuitive interface built with Reflex (Python)
- ⚡ Fast, responsive web experience
- 🏥 Supports informed, budget-conscious healthcare decisions

## 🚀 Getting Started

### Prerequisites

- Python 3.8+
- pip

### Installation

```bash
# Clone the repository
git clone https://github.com/Kriti-here/MediSmart.git
cd MediSmart

# Install system dependencies (if any)
# See apt-packages.txt for system-level packages

# Install Python dependencies
pip install -r requirements.txt

# Run the app
reflex run
```

The app will be available at `http://localhost:3000` by default.
